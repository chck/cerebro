# imageDiff

## Install & Run

```
// node-canvas
$ brew install pkgconfig
$ brew install cairo
$ export PKG_CONFIG_PATH=$PKG_CONFIG_PATH:/opt/X11/lib/pkgconfig

$ xcode-select --install

// gm
$ brew install imagemagick
$ brew install graphicsmagick

$ npm install; npm start
```
